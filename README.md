# Lua-Collections v1.2
 A collections library for Roblox developers.

## Features
#### Data Structures
	LinkedList
	Stack, LinkedStack
	Queue, LinkedQueue
	BinarySearchTree

#### Sorting
	Quick Sort
	Merge Sort
	Tim Sort

#### Searching
	Breadth-First (BFS)
	BinarySearch
	InOrder, PostOrder, PreOrder
